技能库 Skills Hub

一处管理所有 AI agent 技能。把 Claude Code、Codex、各项目的 skill 收进一个本地库,网页上统一开关 ✓ Claude 全局、组合、更新。

GitHub 仓库 → 快速开始

开源 · MIT · 零依赖(Python 标准库) · 纯本地,数据不出机器

# 三行跑起来(macOS / Linux / Windows)
git clone https://github.com/Liang-HZ/skills-hub.git
cd skills-hub
python3 webui.py   # 打开 http://127.0.0.1:7799

它解决什么问题

🗂️

一处管理,处处可用

技能的唯一真源在本地库里,链接到 ~/.claude、~/.codex、~/.agents 或任意项目目录。改一处、处处生效;关掉开关只是摘链接,技能删不丢。

🧩

组合一键开关

把常一起用的技能存成一组,新项目一键开整组,收尾一键关掉。每次变更自动进本地 git 历史,步步可回退。

🔍

第三方来源,主权在你

别人的技能仓库先克隆进隔离区,挑着引入"当时内容的快照"。更新要两次明确授权:先检查看差异,拿到一次性令牌,再更新到你看过的那个版本。

边界声明:它是管理器,不是安全扫描器

  • 它不判断技能是否安全,不调用模型审核,永不执行技能自带的脚本、安装器、示例。
  • 它不自动下载任何东西——所有联网动作(下载来源 / 检查更新 / 执行更新)都在明确标注"联网"的按钮后面,只在你点击时发生。
  • 它把差异和来历摆给你看,决定权在你:第三方技能引入、开启前请自行阅读内容。
  • 实现层防护:写 API 后端强制校验(loopback + 同源 + CSRF 令牌);管理器发起的 git 一律禁用 hook。
